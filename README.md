# 🌷 TULIP – Hand Gesture Controlled Flower Bloom

TULIP is an interactive web experience that allows users to control a blooming flower using real-time hand gestures through their webcam. The project uses MediaPipe Hands for hand tracking and JavaScript to create an engaging, gesture-based interaction.

## ✨ Features

- 🌸 Real-time flower bloom animation
- ✋ Hand gesture recognition using MediaPipe Hands
- 📷 Live webcam integration
- 🌬️ Interactive wind effect through hand movement
- 💻 Responsive full-screen experience
- ⚡ Smooth animations using HTML, CSS, and JavaScript

## 🎮 Controls

- 🤏 **Left hand pinch** → Controls the flower bloom
- 🤏 **Right hand pinch** → Controls flower growth
- 👋 **Move hands** → Creates a wind effect

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- MediaPipe Hands
- HTML Canvas API
